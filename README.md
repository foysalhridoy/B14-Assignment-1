## 📖 Project Overview

> This project is **Assignment-1** of the *AI-Driven Full Stack Web Engineering* course at **Programming Hero**.

The goal was to convert a static design (screenshot) into a clean, semantic, and fully responsive HTML/CSS landing page for a fictional developer conference — **DevConf 2026**.

The build followed a real development workflow: start from a base layout → compare the live output against the design reference → fix mismatches in content, spacing, images, and structure — step by step, the way a developer actually works during a hands-on practice sprint.

## 🔧 Fixes & Improvements Made During Development

While comparing the code against the target design, the following corrections were made — a realistic snapshot of iterative, beginner-friendly debugging:

- ✅ Corrected speaker name typos and track labels to match the design
- ✅ Updated navbar links to match the final design (`Speakers`, `Schedule`, `Tracks`, `Venue`, `Blog`)
- ✅ Added missing price subtext (`per person`, `up to 10 people`) under each pricing card
- ✅ Replaced inline SVG placeholders with simple `img src` paths for easy local image swapping
- ✅ Added a background image to the hero banner
- ✅ Replaced Font Awesome logo icons with real `img` logo references in navbar & footer
- ✅ Fine-tuned logo and image sizing for better visual balance
